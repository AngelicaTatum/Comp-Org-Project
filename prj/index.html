A SimpleS Computer Simulator

This simulator is based on the Java applet by Barry G. Adams of Laurentian University. Several features (Description, Source) were added by Curt Hill of Valley City State University, who also provided the vintage background. Cay Horstmann reimplemented the simulator in JavaScript.

Mnemonic Code (1st word) Code (2nd word) Description
CON 160 n Store the value into a location n
SAVEX 161 n Push values into associated into a value n
ADDIN 44 n Add previous value into variable
SUBIN 45 n Subtract previous value
ADDL 50 n Add current value to variable AX
SUBL 51 n Subtract current value from AX
MULl 52 n Multiply current value from AX n
DIVL 53 n Divide current value from AX n
LEAPN 127 n Jump to instruction in current location
LEAPZ 128 n If AX >= 1
STOP 0 If AX = -1
